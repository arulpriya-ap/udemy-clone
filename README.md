# Udemy Clone (HTML & CSS)

This project is a simple **Udemy website clone** built using **HTML and CSS**.  
It recreates the basic layout of the Udemy homepage and demonstrates fundamental web development concepts.

## 🚀 Features
- Created a structured webpage using **HTML**
- Implemented multiple **div containers** for layout organization
- Designed **Header, Body, and Footer sections**
- Applied **CSS styling** to enhance the UI
- Used **Flexbox** to align and arrange elements efficiently
- Added course cards and navigation similar to the Udemy interface

## 🛠 Technologies Used
- **HTML5**
- **CSS3**
- **CSS Flexbox**

## 📚 What I Learned
- Structuring webpages with semantic HTML
- Styling layouts using CSS
- Using **Flexbox** for responsive alignment
- Organizing components using multiple div containers
